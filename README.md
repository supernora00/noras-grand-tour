# Nora’s Grand Tour · Journey OS Alpha 2

A mobile-first travel companion for Europe 2026. It works on GitHub Pages and can also be opened locally by double-clicking `index.html`.

## Alpha 2 changes

- Added `Travel Phrases` with Chinese, English, French, German and Italian.
- Added one-tap pronunciation through the browser Speech Synthesis API.
- Added local-compatible copy and search.
- Added Lauterbrunnen and Staubbach Falls to Day 7, Trip and Maps.
- Updated Day 7 to use the Jungfraujoch → Kleine Scheidegg → Wengen → Lauterbrunnen loop.
- Archive no longer depends on `fetch()`, so it works under `file://`.
- Retained the existing Home / Trip / Wallet / Archive / More navigation.

## Deployment

Upload all files and folders inside this directory to the root of the GitHub repository, replacing the previous version.
